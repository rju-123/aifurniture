# 技术问题咨询
不用实现代码，是技术问题的咨询。
我想完成的AI功能是，将两个图片进行指定要求的如何，具体而言用户上传客厅图片和沙发图片，并且用户拖拽的方式将沙发拖到客厅的图片上,以表明想将沙发具体摆放的位置，然后大模型，就把对应的沙发自然的融入到房间中对应的位子。

好多现有的大模型工具，都不是特别满足我的需求，比如：
1. 方案一：用户上传客厅图片和遮图，遮图以黑白表示，白色部分可编辑，以选定重构的位置，但重构的位置是是通过文本的方式描述，这样就丢失沙发的形态特征。
2. 方案二：用户上传客厅图片和沙发图片，但无法通过文字准确的描述具体的安放位置。
3. 方案三：另外了解有几个技术，类似方案一，沙发的特性是通过图片传输的，但没有找到相关的一个具体可在线调用的服务。
4. 方案四：在方案一上做改进，将在客厅的图片上，可编辑的部分使用浅浅的透明的绿色，然后再上传一个沙发的图片，通过prompt告诉大模型将沙发融合到绿色的部分，但是效果来说，大模型总是将沙发绘制到我指定范围之外很多。

我现在陷入到难题，难以找到一个可用的方案，帮我思考一个可行方案。
