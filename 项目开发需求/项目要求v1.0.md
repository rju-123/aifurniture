# 项目大版本迭代
我计划对智能家装的方式进行重构，具体的方案如下：
1. 用户仍然是上传客厅图片，但家具的选择限定在一个。
2. 用户不是直接将家具拖拽到指定的位置，而是通过大画笔涂画的方式在图片上客厅图片上涂一层浅浅的蓝色，表示家具要放置的位置，这个涂画的过程有点像绘图工具中的橡皮擦，不过这个不是清除图像而只是将图像对应的涂上浅浅蓝色，这个蓝色是透明的仍然可以看到蓝色后的图片。这个涂画支持回退一键清除已涂改的地方。用户点击“下一步”按钮后，保存这个附带蓝色的图片，这个是叫做客厅mask图片。
3. 用户将mask图片和家具图传给千问图片大模型，具体的代码参考 @test_qwen_image_fusion2.py
